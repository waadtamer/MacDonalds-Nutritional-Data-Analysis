# McDonald’s Menu Nutritional Data Analysis (Tableau) 📊🍔

An exploratory data visualization project that dives into the nutritional reality of fast food using a McDonald's menu dataset. Through interactive Tableau dashboards, this project uncovers hidden nutritional patterns, analyzes dietary balance, and highlights key indicators for health-conscious choices and diabetes management
📌 Key Analysis Areas
Fat Composition Analysis:** Explores the relationship between different food categories and their fat types (Saturated, Trans fats, etc.).
Nutritional Content Overview:** Compares average nutritional values across menu categories to highlight macro and micronutrient patterns.
Diabetes Management Awareness:** Evaluates glycemic and lipid content to highlight crucial nutritional indicators for individuals managing diabetes.
Dietary Balance:** Analyzes the Protein-to-Calorie ratio across items to identify options that provide the best nutritional efficiency.
 Highlighted Insight (The Outlier)
Through scatter plot visualization, a significant anomaly was detected:
Item: Chicken McNuggets (40 pieces) - *Chicken & Fish Category*
Metrics:1,880 Calories / ~87g portion size.
Finding:This item dramatically deviates from the entire menu average, serving as a powerful example of how data visualization instantly reveals critical data outliers.
